## 📁 Acessar Projeto Localmente

- *Clonar o repositório:*

```
$ git clone git@github.com:mathews-r/trybewallet.git
```

- *Acessar o projeto trybewallet:*

```
$ cd trybewallet
```

- *Instalar dependências:*

```
$ npm install
```

- *Executar projeto:*

```
$ npm start
```
- *Executar os testes do projeto:*

```
$ npm run test
```
